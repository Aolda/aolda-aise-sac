## 3. Product Info

### a. 프로젝트 개요
> 🎤 어떤 이유에서 시작된 프로젝트였나요?

기존까지는 아올다 클라우드를 구현하기 위한 소프트웨어 개발작업 및 인프라 구축작업 위주로 진행되어왔습니다.
하지만 실제 운영환경으로 발전하며 심각한 취약점으로 발전할 수 있는 위협들이 존재했고, 이를 보완할 수 있는 기반이 필요했습니다.
따라서 코드로 관리되는 보안기반을 구현하고 향후 아올다 대수에서 편리하게 보안환경을 관리/적용할 수 있도록 하고자 본 프로젝트를 기획/진행하게 되었습니다.

### b. 주요 기능
> 🎤 이 제품은 어떤 기능들을 제공하나요?

- Ansible 기반으로 작성되며, 각 tool이 단일 제품의 보안을 담당합니다.
- 하위 task를 통해 해당 제품에 적용할 보안정책 별 관리를 진행하며, 파일 내부에는 해당 정책의 적용을 위한 상세 적용방법을 설명/관리합니다.
