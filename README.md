# aolda-aise-sac

>: Aolda - AISE (Aolda Infrastructure Security Enhancement) 프로젝트

- Ansible 기반 Security as Code 프로젝트 레포지토리입니다.


## Repository Structure

- docs/        : 문서 (규칙, 가이드 등)
- inventory/   : Ansible 인벤토리
- playbooks/   : 실행용 playbook
- roles/       : 보안 설정 Ansible Role
- group\_vars/  : 그룹 변수


## Workflow

- main: 안정화된 기준 브랜치

- feature/docs/chore 브랜치에서 작업 후 PR → main merge
